# OpenClaw VPS

## Overview

**OpenClaw VPS** is a managed hosting platform for OpenClaw, a private AI assistant that runs on your own infrastructure. The service provides zero-DevOps deployment, enterprise security, and predictable monthly pricing for users who want their own AI assistant without the technical overhead.

Live at: [openclawvps.com](https://openclawvps.com)

## Project Scale

- **Platform**: Service business with web application
- **Tech Stack**: Next.js + TypeScript + Convex + Polar
- **Industry**: AI Infrastructure / Managed Services
- **Architecture**: Serverless with real-time database
- **Target**: Non-technical users wanting private AI assistants

## Problem Solved

Running a self-hosted AI assistant requires significant technical expertise:

- **Infrastructure complexity**: VPS provisioning, tunnel configuration, security hardening
- **Ongoing maintenance**: Security patches, updates, monitoring
- **Configuration challenges**: Messaging app integrations, skill setup
- **DevOps overhead**: Most users want AI benefits without server management
- **Support gaps**: Self-deploy users often get stuck without help

## The Solution

OpenClaw VPS offers two paths to get running:

### Done-For-You Setup

- **Standard ($500)**: Discovery call, full deployment, 30-day support, 1 month Pro hosting
- **Premium ($1,000)**: Extended setup, custom integrations, 3-5 skills configured, training call, 3 months Business hosting

### Managed Hosting Plans

- **Starter ($39/mo)**: 1 messaging app, basic monitoring, 48hr support
- **Pro ($69/mo)**: 3 apps, weekly health checks, config changes, 24hr support
- **Business ($129/mo)**: Unlimited apps, dedicated infrastructure, same-day support

## Technical Architecture

### Frontend (Next.js)

Modern React application with:

- **App Router**: Server components and actions
- **TypeScript**: Full type safety throughout
- **Tailwind CSS**: Responsive, themeable design
- **Dark mode**: System preference detection

### Backend (Convex)

Real-time serverless infrastructure:

- **Bookings system**: Track checkout status and customer data
- **Survey collection**: Post-signup feedback
- **Real-time sync**: Instant updates across dashboard
- **Type-safe queries**: End-to-end TypeScript types

### Payments (Polar)

Checkout and subscription management:

- **One-time payments**: Done-for-you setup packages
- **Recurring subscriptions**: Monthly hosting plans
- **Webhook integration**: Real-time payment status updates
- **Customer management**: Email-based identification

### Security (Turnstile)

Bot protection on checkout:

- **Cloudflare Turnstile**: Privacy-preserving CAPTCHA
- **Rate limiting**: Per-email checkout throttling
- **Zod validation**: Server-side input sanitization
